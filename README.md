# منصّة إدارة المشاريع — علامة (Alaamaa-PMO)

داشبورد حوكمة المشاريع بين مكتب إدارة المشاريع (PMO) في علامة وعملائها.

- **النشر:** Cloudflare Pages على `pmo.alaamaa.com`
- **قاعدة البيانات والمصادقة:** Supabase (مشترك مع منصة الأدمن، جداول `pmo_` منفصلة)
- **الدخول:** Google (حساب علامة)

## الصلاحيات
- PMO (admin): بناء وتعديل الخطة، اعتماد Baseline، بتّ طلبات التغيير.
- فريق التسويق (manager): تحديث الحالة والتقدّم.
- العميل: عرض مشروعه فقط (لوحة قيادة، جانت، مخرجات).

## الملف الرئيسي
`index.html` — نواة المرحلة 1 (دخول + لوحة قيادة + جدول MS Project + جانت).

راجع `ROADMAP.md` للمراحل القادمة.
